## History of changes for WinPython 3.4.1.1

The following changes were made to WinPython distribution since version 3.4.1.0.

### Tools

Removed packages:

  * [MinGW32](http://www.mingw.org) 4.8.1 ((32-bit version only) Windows specific header files and import libraries, C/C++ and Fortran compilers)

### Python packages

New packages:

  * [ViTables](http://vitables.org) 2.1 (Graphical tool for browsing and editing files in both HDF5 and PyTables formats)
  * [brewer2mpl](http://pypi.python.org/pypi/brewer2mpl) 1.4 ()
  * [ggplot](https://github.com/yhat/ggplot) 0.6.5 (ggplot for python)
  * [ipython-sql](http://ipython.org) 0.3.3 (Enhanced Python shell)
  * [joblib](http://pypi.python.org/pypi/joblib) 0.8.2 ()
  * [julia](http://pypi.python.org/pypi/julia) 0.1.1.2 ()
  * [libpython](http://pypi.python.org/pypi/libpython) 1.0 ()
  * [mpld3](http://pypi.python.org/pypi/mpld3) 0.2 ()
  * [mysql-connector-python](http://pypi.python.org/pypi/mysql-connector-python) 1.2.2 ()
  * [pep8](http://pypi.python.org/pypi/pep8) 1.5.7 ()
  * [pg8000](http://pypi.python.org/pypi/pg8000) 1.9.14 ()
  * [pyflakes](http://pypi.python.org/pypi/pyflakes) 0.8.1 ()
  * [pyodbc](http://pypi.python.org/pypi/pyodbc) 3.0.7 ()
  * [reportlab](http://www.reportlab.org) 3.1.8 (The PDF generation library)
  * [rope_py3k-0.9.4](http://pypi.python.org/pypi/rope_py3k-0.9.4) 1 ()
  * [rpy2](http://pypi.python.org/pypi/rpy2) 2.4.3 ()
  * [seaborn](http://pypi.python.org/pypi/seaborn) 0.3.1 ()
  * [sqlite_bro](https://github.com/stonebig/sqlite_bro) 0.8.6 (a graphic SQLite Client in 1 Python file)

Upgraded packages:

  * [Pillow](http://pypi.python.org/pypi/Pillow) 2.4.0 → 2.5.2 (Python Imaging Library (fork))
  * [PyOpenGL](http://pyopengl.sourceforge.net) 3.1.0b3 → 3.1.0 (Cross platform Python binding to OpenGL and related APIs)
  * [SQLAlchemy](http://www.sqlalchemy.org) 0.9.6 → 0.9.7 (SQL Toolkit and Object Relational Mapper)
  * [astroid](http://pypi.python.org/pypi/astroid) 1.1.1 → 1.2.0 (Rebuild a new abstract syntax tree from Python's ast (required for pylint))
  * [docutils](http://docutils.sourceforge.net) 0.11 → 0.12 (Text processing system for processing plaintext documentation into useful formats, such as HTML or LaTeX (includes reStructuredText))
  * [ipython](http://ipython.org) 2.1.0 → 2.2.0 (Enhanced Python shell)
  * [mahotas](http://pypi.python.org/pypi/mahotas) 1.1.0 → 1.2.1 (Computer Vision library)
  * [numpy-MKL](http://numpy.scipy.org/) 1.8.1 → 1.8.2 (NumPy: multidimensional array processing for numbers, strings, records and objects (SciPy's core module))
  * [patsy](http://pypi.python.org/pypi/patsy) 0.2.1 → 0.3.0 (Describing statistical models using symbolic formulas)
  * [pylint](http://www.logilab.org/project/pylint) 1.2.1 → 1.3.0 (Logilab code analysis module: analyzes Python source code looking for bugs and signs of poor quality)
  * [scikit-image](http://pypi.python.org/pypi/scikit-image) 0.10.0 → 0.10.1 (Image processing toolbox for SciPy)
  * [scikit-learn](http://pypi.python.org/pypi/scikit-learn) 0.15.0b1 → 0.15.1 (A set of Python modules for machine learning and data mining)
  * [setuptools](http://pypi.python.org/pypi/setuptools) 3.8.1 → 5.4.2 (Download, build, install, upgrade, and uninstall Python packages - easily)
  * [simplejson](http://pypi.python.org/pypi/simplejson) 3.5.2 → 3.6.2 (Simple, fast, extensible JSON (JavaScript Object Notation) encoder/decoder)
  * [tornado](http://pypi.python.org/pypi/tornado) 3.2.2 → 4.0.1 (Scalable, non-blocking web server and tools (required for IPython notebook))

Removed packages:

  * [PySide](https://pypi.python.org/pypi/PySide) 1.2.2 (Python binding of the cross-platform GUI toolkit Qt)
