## History of changes for WinPython 3.3.5.1

The following changes were made to WinPython distribution since version 3.3.3.1.

### Python packages

New packages:

  * [ViTables](http://vitables.org) 2.1 (Graphical tool for browsing and editing files in both HDF5 and PyTables formats)
  * [pep8](http://pypi.python.org/pypi/pep8) 1.5.7 ()
  * [pyflakes](http://pypi.python.org/pypi/pyflakes) 0.8.1 ()
  * [reportlab](http://www.reportlab.org) 3.1.8 (The PDF generation library)
  * [rope_py3k-0.9.4](http://pypi.python.org/pypi/rope_py3k-0.9.4) 1 ()

Upgraded packages:

  * [Pillow](http://pypi.python.org/pypi/Pillow) 2.5.1 → 2.5.2 (Python Imaging Library (fork))
  * [Python](http://www.python.org/) 3.3.3 → 3.3.5 (Python programming language with standard library)
  * [astroid](http://pypi.python.org/pypi/astroid) 1.1.1 → 1.2.0 (Rebuild a new abstract syntax tree from Python's ast (required for pylint))
  * [ggplot](https://github.com/yhat/ggplot) 0.6.0 → 0.6.5 (ggplot for python)
  * [ipython-sql](http://ipython.org) 0.3.1 → 0.3.3 (Enhanced Python shell)
  * [julia](http://pypi.python.org/pypi/julia) 0.1.1 → 0.1.1.2 ()
  * [numpy-MKL](http://numpy.scipy.org/) 1.8.1 → 1.8.2 (NumPy: multidimensional array processing for numbers, strings, records and objects (SciPy's core module))
  * [pg8000](http://pypi.python.org/pypi/pg8000) 1.9.10 → 1.9.14 ()
  * [pylint](http://www.logilab.org/project/pylint) 1.2.1 → 1.3.0 (Logilab code analysis module: analyzes Python source code looking for bugs and signs of poor quality)
  * [rpy2](http://pypi.python.org/pypi/rpy2) 2.4.2 → 2.4.3 ()
  * [setuptools](http://pypi.python.org/pypi/setuptools) 3.8.1 → 5.4.2 (Download, build, install, upgrade, and uninstall Python packages - easily)
  * [simplejson](http://pypi.python.org/pypi/simplejson) 3.6.0 → 3.6.2 (Simple, fast, extensible JSON (JavaScript Object Notation) encoder/decoder)
  * [sqlite_bro](https://github.com/stonebig/sqlite_bro) 0.8.5 → 0.8.6 (a graphic SQLite Client in 1 Python file)
  * [tornado](http://pypi.python.org/pypi/tornado) 4.0 → 4.0.1 (Scalable, non-blocking web server and tools (required for IPython notebook))

Removed packages:

  * [PySide](https://pypi.python.org/pypi/PySide) 1.2.2 (Python binding of the cross-platform GUI toolkit Qt)
